## Altimetry

### Transition Altitude 

In the New Caledonia sector, the transition altitude is 11,000 ft.

### Transition Level

| QNH (hPa)               | Transition Level |
| -------------------| ---------------- | 
| Less than 978 | FL 140 | 
| Between 978 to 1012 | FL 130 | 
| Between 1013 to 1048 | FL 120 | 
| Greater than 1048 | FL 110 |
