## Python Dependencies

The following Python packages are used in this project:

- **MkDocs**  
  Static site generator.  
  License: [BSD 2-Clause License](https://github.com/mkdocs/mkdocs/blob/master/LICENSE)  
  Source: [MkDocs Website](https://www.mkdocs.org/) | [GitHub Repository](https://github.com/mkdocs/mkdocs)

- **mkdocs-material**  
  A Material Design theme for MkDocs.  
  License: [MIT License](https://github.com/squidfunk/mkdocs-material/blob/master/LICENSE)  
  Source: [GitHub Repository](https://github.com/squidfunk/mkdocs-material)

- **mkdocs-awesome-pages-plugin**  
  Simplifies configuring page titles and their order in MkDocs.  
  License: [MIT License](https://github.com/lukasgeiter/mkdocs-awesome-pages-plugin/blob/master/LICENSE.md)  
  Source: [GitHub Repository](https://github.com/lukasgeiter/mkdocs-awesome-pages-plugin)

- **mkdocs-git-revision-date-localized-plugin**  
  Adds a localized "Last updated" date to your MkDocs pages.  
  License: [MIT License](https://github.com/timvink/mkdocs-git-revision-date-localized-plugin/blob/master/LICENSE)  
  Source: [GitHub Repository](https://github.com/timvink/mkdocs-git-revision-date-localized-plugin)

- **mkdocs-redirects**  
  Handles page redirections in MkDocs.  
  License: [MIT License](https://github.com/datarobot/mkdocs-redirects/blob/master/LICENSE.md)  
  Source: [GitHub Repository](https://github.com/datarobot/mkdocs-redirects)

- **mkdocs-glightbox**  
  Adds lightbox functionality for images in MkDocs.  
  License: [MIT License](https://github.com/blueswen/mkdocs-glightbox/blob/main/LICENSE)  
  Source: [GitHub Repository](https://github.com/blueswen/mkdocs-glightbox)

## GitHub Actions Workflows

The following GitHub Actions workflows are used for automation in this project:

### CI Workflow

This workflow automates the deployment of the MkDocs-based site to GitHub Pages.

- **actions/checkout@v2**  
  Checks out the repository.  
  License: [MIT License](https://github.com/actions/checkout/blob/main/LICENSE)  
  Source: [actions/checkout GitHub Repository](https://github.com/actions/checkout)

- **actions/setup-python@v2**  
  Sets up a Python environment.  
  License: [MIT License](https://github.com/actions/setup-python/blob/main/LICENSE)  
  Source: [actions/setup-python GitHub Repository](https://github.com/actions/setup-python)

### Discord Notification Workflow

This workflow sends a notification to Discord when a pull request is merged and labeled `notify`.

- **Ilshidur/action-discord@master**  
  Sends a notification to Discord via webhook.  
  License: [MIT License](https://github.com/Ilshidur/action-discord/blob/master/LICENSE)  
  Source: [Ilshidur/action-discord GitHub Repository](https://github.com/Ilshidur/action-discord)
